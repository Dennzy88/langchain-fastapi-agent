## 🚀 **Quick Start with Docker**

```bash
# 🐳 One-command Docker deployment
git clone https://github.com/Dennzy88/langchain-fastapi-agent.git
cd langchain-fastapi-agent
cp .env.example .env
# Add your OpenAI API key to .env
./deploy_docker.sh
```
